# fluxa

Fluxa - Akıllı Fatura ve Abonelik Yönetim Sistemi
Bu proje, fatura takibi ve abonelik yönetimi süreçlerini otomatize etmek amacıyla geliştirilmiş bir Full-Stack uygulamadır.
Proje Hakkında
Açıklama: Flutter ile geliştirilen mobil arayüz, React tabanlı web yönetim paneli ve Node.js altyapısı ile uçtan uca fatura yönetim sistemi.
Ekran Görüntüleri: https://quiver-bone-d7d.notion.site/Mobil-Abonelik-Ve-Veritaban-Y-netim-38722ed63b75800193f1ea9967c73955
 Kullanılan Teknolojiler
Mobil: Flutter (Dart)
Web: React.js
Backend: Node.js, PM2
Veritabanı/Servis: Firebase
